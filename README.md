\# So hüpft der Hase



KI-Projekt mit A\*-Algorithmus.



\## Funktionen



\- A\* Pathfinding

\- Manhattan Heuristik

\- Mehrere Hasen

\- Level-System

\- Automatische Wegplanung



\## Steuerung



Pfeiltasten = Bewegung



SPACE = Aktuellen Hasen lösen



A = Alle Hasen automatisch lösen



R = Neustart



ESC = Menü
